# Architecture Decision Records (ADR)

Folder for architectural decisions. **ADR canonical text is English** (`docs/decisions/`). **Russian translations** for reading: `docs/decisions/ru/` (same structure, linked from EN).

---

## ADR template

```markdown
# ADR-NNN: Title

**Status:** proposed | accepted | deprecated  
**Date:** YYYY-MM-DD

## Context

What problem or question we address.

## Decision

What we decided.

## Consequences

Pros, cons, what we deferred.

## Links

- Concept: docs/concept/en/...
- Source (RU detail): docs/concept/ru/...
- Schema: docs/concept/schemas/...
```
